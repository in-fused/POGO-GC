How spawn timers work in this app: Timers are simulated using real PoGo spawn mechanics — most points cycle every hour on the same minute, spawns last 15–30 min. There is no official Niantic spawn API. Real-time data requires community scanners. See below for everything available.

Free Public APIs

pogoapi.net
FREE
Game master data: current raid bosses, nesting Pokemon list, shiny rates, type chart, community day, weather boosts, egg groups. Updated direct from Niantic's game master file. Best free source for live raid boss list and nest eligibility.
pogoapi.net/api/v1/current_raid_bosses.json

Community / Crowdsourced

PogoMap.info
CROWD
Worldwide crowdsourced map of PokéStops, Gyms, Team Rocket invasions, nests, power spots, S2 cells. No open JSON API but embeds on pages. Best reference for stop/gym coordinates per city.

Nest Atlas — The Silph Road
CROWD
Community-confirmed nest reports worldwide. Nests rotate every 2 weeks (Thursday 01:00 UTC). No public JSON API but web-accessible. Best for confirming if a local park is a real nest.

Official Niantic Spawn API
DOES NOT EXIST
Niantic has never released a public spawn API. All "real-time" spawn maps (PokéVision etc.) used reverse-engineered calls and were shut down or C&D'd. The timers in this app are simulated using seeded RNG matching real PoGo spawn point mechanics (hourly cycles, 15–30 min windows).
